# RosNext

RosNext is a starter template for ROS 2 projects with a focus on structure, modularity, and developer experience.

## Included official example package

This template includes `heartbeat_node` as an official RosNext starter package.

Purpose:
- validate the workspace build
- demonstrate package structure
- show parameter loading with YAML
- provide a reusable node example

Location:
- `src/communication/heartbeat_node`
